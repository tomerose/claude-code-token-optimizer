# Sponsor QR Codes

Replace the placeholder files in this directory with your actual payment QR codes:

| File | Source |
|------|--------|
| `wechat-qr.png` | WeChat → Me → Pay → QR Code → Save Image |
| `alipay-qr.png` | Alipay → Receive → Save QR Code |

**Image specs:** PNG format, 400×400px, <200KB
